# 🔥 BABЛО - AI Crypto Trading Bot with 90% Prediction Accuracy

**Автоматическая система торговли криптовалютами с предсказанием на 90% точности**

## 🎯 Особенности

- 🤖 **AI Prediction Engine** - GPT-4 + локальные модели (Llama, Mistral)
- 📊 **Multi-Source Data Collection** - Bybit, Binance, CoinGecko, Reddit, Twitter, GitHub, Whale Trackers
- 📈 **Advanced Analytics** - Technical + Fundamental + Sentiment Analysis
- 💰 **Risk Management** - До 50x плеча с автоматическим управлением
- 🎓 **Strategy Optimization** - Автоматический подбор оптимальных стратегий
- ⚠️ **Smart Alerts** - Визуальные сигналы с возможностью включения/отключения
- ✅ **Confirmation System** - Требует подтверждения перед каждой сделкой
- 🔐 **Security First** - Шифрование API ключей, безопасное хранение

## 📋 Stack

**Backend:**
- Python 3.11
- FastAPI + Uvicorn
- SQLAlchemy + PostgreSQL
- OpenAI API + Ollama (локальные модели)
- Async/Await для высокой производительности

**Frontend:**
- React 18
- TailwindCSS
- Chart.js для графиков
- WebSocket для реал-тайм данных

## 🚀 Быстрый старт

```bash
# Запуск с Docker
docker-compose up -d

# Доступ
Frontend: http://localhost:3000
Backend API: http://localhost:8000
API Docs: http://localhost:8000/docs
```
